
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript.
This is a base api with Nest and Typescript (User autenticacion,local-paspport, jwt-passport, sequelize, api documentation and nodemailer service)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
